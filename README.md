> A deep learning model that detects whether a person is wearing a face mask or not using Convolutional Neural Networks (CNN).

---

## 📌 Overview

This project uses a CNN-based image classification model to detect face masks in real-time. Trained on labeled images of people with and without masks.

---

## ✨ Features

- ✅ Binary classification: Mask / No Mask
- ✅ Built with CNN using TensorFlow/Keras
- ✅ Real-time detection capability
- ✅ High accuracy on test data

## 📊 Model Architecture

- Input Layer → Conv2D → MaxPooling
- Conv2D → MaxPooling
- Flatten → Dense → Dropout
- Output: Sigmoid (Binary Classification)
